# AI MODEL LAB

Interaktywne laboratorium uczenia maszynowego i odpowiedzialnej sztucznej inteligencji dla edukacji.

Uczniowie trenują rzeczywisty model drzewa decyzyjnego, sprawdzają, czego nauczył się z danych, wykrywają nierówne traktowanie, przeprowadzają testy kontrfaktyczne oraz obserwują wpływ własnych zmian na wyniki modelu.

## Aplikacja

https://aimodellab.github.io/

## Wersja konkursowa

AI MODEL LAB 2.4

## Najważniejsze elementy

- rzeczywisty trening drzewa decyzyjnego w przeglądarce,
- analiza zbioru uczącego,
- audyt nierównego traktowania,
- testy kontrfaktyczne,
- wybór i ocena naprawy modelu,
- porównanie V1 i V2,
- raporty PDF i HTML,
- rozszerzone Wyzwanie badawcze.

## Prywatność

Dane są fikcyjne. Model działa lokalnie w przeglądarce — bez konta i wysyłania danych.
